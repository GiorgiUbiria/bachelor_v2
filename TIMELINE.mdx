# Bachelor's Project Timeline - ML in Everyday Applications

## üéØ Project Overview

**Title**: "Machine Learning in Everyday Digital Experiences: A Practical Demonstration"

**Objective**: Create an e-commerce platform demonstrating ML algorithms in daily digital interactions including trend analysis, recommendations, dynamic pricing, enhanced search, and chatbots.

### Core ML Features:
1. **Trend Analysis & Forecasting** - Sales trends and seasonal patterns
2. **Recommendation System** - Personalized product suggestions  
3. **Dynamic Pricing** - Real-time price optimization
4. **Enhanced Site Search** - ML-powered search relevance
5. **Intelligent Chatbot** - Customer service automation

### Technology Stack:
- **Frontend**: React 19, Vite, TailwindCSS, Zustand, React Router, Axios, Zod
- **Backend**: Go (Fiber v3), PostgreSQL
- **ML Service**: Python, FastAPI, scikit-learn, pandas, numpy
- **Infrastructure**: Docker, Nginx
